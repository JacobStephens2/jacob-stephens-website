Posts

I completed LinkedIn Learning's Become a DevOps Engineer learning path.

Motivation

I experienced continuous deployment (CD) through Gatsby Cloud, and I became hooked. I wanted to learn how to implement CD with web apps not hosted on Gatsby Cloud. I also completed LinkedIn Learning's Become a Full-Stack Web Developer learning path, and the final course in that path was DevOps Foundations. I learned that CD is embraced by DevOps, so I thought learning more about DevOps would give me more insight on how to implement CD. I also saw in Stack Overflow's 2021 Developer Survey that DevOps specialists do quite valuable work. These factors combined to encourage me to learn more about DevOps and take the DevOps learning path.
